1. Overview

1.1 Game Concept

Brainrot Blast is a casual Match-3 puzzle game featuring Italian brainrot meme characters. Players match colorful brainrot icons on a grid to score points, complete levels, and collect brainrot pets/badges as rewards.

1.2 Target Platform

  • Android (Google Play)
  • iOS (App Store)

1.3 Target Audience

  • Casual gamers
  • Gen Alpha / Gen Z (brainrot meme fans)
  • Ages 8+

1.4 Core Pillars

  1. Simple & Casual - Easy to learn, play anywhere
  2. Collectible - Unlock brainrot characters as rewards
  3. Infinite Replayability - Procedurally generated levels
  4. Offline First - No internet required to play

2. Gameplay

2.1 Core Mechanic

Classic Match-3 gameplay:

  • Swap adjacent tiles to create matches of 3+ identical icons
  • Matched tiles disappear, new tiles fall from above
  • Create combos for bonus points
  • Complete objectives within limited moves

2.2 Controls

InputAction
SwipeSwap two adjacent tiles
TapSelect tile (optional: tap another to swap)

2.4 Procedural Level Generation

Levels are algorithm-driven, not manually designed:

Parameters that scale with level number:
- Grid size: 5x5 → 6x6 → 7x7 (progressive)
- Move limit: Decreases as levels increase
- Objective difficulty: Increases targets
- Blocker frequency: More obstacles at higher levels
- Icon variety: 5 types → 6 types → 7 types

Difficulty Curve Algorithm

2.5 Special Tiles & Power-ups

Match TypeResult
Match 4 (line)Line Blast - clears entire row/column
Match 4 (square)Bomb - clears 3x3 area
Match 5Rainbow - clears all of one icon type
Match 6+Super Bomb - clears 5x5 area

Blast Animation

When player completes a level:
  • Screen fills with particle explosion
  • Matched icons fly outward
  • "BLAST!" text animates
  • Confetti particles rain down
  • Satisfying sound effect plays
